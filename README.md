# Quiosco App

Aplicación que gestiona los productos de un quiosco de comida tanto por parte de los usuarios como en cocina

## Principales características del proyecto

- API REST
- Context API
- Filtrado
- Hooks
- Tailwind 
- JavaScript
- Next.js
- Axios
- SWR
- Prisma
- SQL
